# Patient manager

Manage a list of patients.

Patients contain :

- Name
- Surname
- Age
- Date of Birth

## Current stack

- VueJs + BootstrapVue for the Bootstrap-Vue
- Express + MongoDB backend

# How to run
 Remarks:
     A data directory will be created locally for the MongoDB instance.

    docker-compose build && docker-compose up
